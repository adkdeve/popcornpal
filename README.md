# 🍿 PopcornPal

PopcornPal is a high-performance, feature-rich movie discovery and tracker mobile application built using Flutter. The app provides user-centric entertainment streams, tailored movie searches, custom interactive favorites lists, and real-time engagement metrics designed to maximize application retention.

## 📱 Features

* **Dynamic Movie Exploration:** Infinite scrolling and low-latency data fetching for trending, top-rated, and upcoming movies.
* **Smart Search & Advanced Filters:** Structural localized indexing allowing users to find content based on genres, release dates, and ratings.
* **Interactive Bookmarks & Favorites:** Custom storage solutions enabling users to manage personalized watchlists on the fly.
* **Engagement Analytics:** Real-time engagement metric counters tracking user interactions to deliver responsive UI updates.
* **Optimized UI Engine:** Engineered with robust state management to reduce UI frame drops and maintain a fluid 60 FPS profile across low-end testing hardware.

## 🛠️ Built With

* **Frontend Framework:** Flutter & Dart
* **State Management:** Provider / GetX (Metric-Driven UI Performance)
* **API Architecture:** RESTful APIs with secure object schemas and response caching
* **Local Storage:** Secure offline-first data synchronization (Drift/SQLite or Hive)
* **UI/UX Design:** Pixel-perfect, responsive component layouts with native-feeling animations
